# Dictionary_from_table_to_doc
Создание печатной версии словаря из табличного представления

Формат табличного словаря
Русско-осетинский
1. rus - русское словарное слово (столбик A)
2. reduct - сокращение, множественное/единственное число, мужской/женский (B)
3. notice -  часть речи, область бот./рел. (C)
4. multy - столбик помечающий многозначные значения, там должен быть номер значения (D)
5. ose - осетинский перевод (E)
6. form - варианты написания (бир. англисæгтæ)/(æвзæрстон) (F)
7. transc - транскрипция осетинского перевода (G)
8. example_rus_1 - пример на русском языке (H)
9. example_ose_1 - перевод на осетинкий примера из предыдущей колонки (I)
10. example_rus_2 - пример на русском языке (J)
11. example_ose_2 - перевод на осетинкий примера из предыдущей колонки (K)
12. example_rus_3 - пример на русском языке (L)
13. example_ose_3 - перевод на осетинкий примера из предыдущей колонки (M)
14. example_rus_4 - пример на русском языке (N)
15. example_ose_4 - перевод на осетинкий примера из предыдущей колонки (O)
всего может быть 10 примеров с 7 по 17 столбик
